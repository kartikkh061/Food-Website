Food Website - A Journey Through the Heart of India
A modern, responsive, and highly interactive frontend web application for an Indian restaurant. This project features a full user experience, from browsing an authentic menu to tracking active orders.

🚀 Live Demo


✨ Features
Interactive Landing Page: A dynamic hero section featuring GSAP animations and Parallax.js effects.

Responsive Menu: A filtered food menu (Breakfast, Lunch, Dinner) with real-time search functionality.

User Authentication: Complete UI flow for Login, Signup, and Forgot Password with form validation and accessibility considerations.

Order Tracking: A dedicated orders.html page to monitor order status and view detailed history.

Smooth Navigation: Custom smooth scrolling and a "sticky" header that adapts as you scroll.

Accessibility First: Includes skip-navigation links, ARIA labels, and dedicated accessibility stylesheets for screen readers.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Neumorphism UI style), JavaScript (ES6).

Frameworks: Bootstrap 4/5.

Libraries:

GSAP: For high-performance animations.

Swiper.js: For touch-responsive sliders (e.g., Tandoori Platter, Team section).

Parallax.js: For depth-based background movements.

Fancybox: For sleek image and video popups.

jQuery: For DOM manipulation and slider initialization.

🎨 Design Notes
This project utilizes a Neumorphic design language, characterized by soft shadows (inner and outer) to create a "soft UI" look. The color palette is centered around #ff8243 (Orange) and #f3f3f5 (Light Gray) to provide a clean yet appetizing aesthetic.
